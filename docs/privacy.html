Privacy Policy

Last Updated: January 2026

1. Data Collection

OpenCapture is a local-first application. We do not collect, store, or transmit any personal data, images, or usage statistics to any external servers.

2. Local Storage

All screenshots and recordings are stored locally on your device in the location you select.

3. Updates

The application may check GitHub Releases for updates if you manually navigate to the download page, but the app itself does not auto-update or phone home.

4. Contact

If you have any questions, please contact the developer via the GitHub repository.
